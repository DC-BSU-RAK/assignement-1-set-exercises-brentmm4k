The h1 to h6 tags levels are the different levels of the section headings

The p tag is usually used for long form text and does not have different levels of font size like the h1-h6 tags.
The br tag is used to make a new line
There are multiple ways to style texts without using a style sheet such as b tag for bold, i tag for italic, u tag for underline, sub tag for superscript, and finally, the sub tag for subscript
To make a list, the use of the li tag and either the ol, and ul tag

Example ul and li for bulletin lists:

  • Section 1
  • Section 2
  • Section 3

Example ol and li for numbered lists:

  1. Section 1
  2. Section 2
  3. Section 3
